편집 명령어

Ctrl + Z : 실행 취소
Ctrl + Shift + Z : 재실행
Ctrl + X : 잘라내기
Ctrl + C : 복사
Ctrl + V : 붙여넣기
Ctrl + Shift + V : 같은 위치에 붙여넣기

오브젝트 조작

Ctrl + G : 그룹화
Ctrl + Shift + G : 그룹 해제
Ctrl + ] : 앞으로 보내기
Ctrl + [ : 뒤로 보내기
Ctrl + Shift + ] : 맨 앞으로
Ctrl + Shift + [ : 맨 뒤로

보기 조작

Ctrl + 0 : 전체 화면에 맞춤
Ctrl + 1 : 100% 크기
Ctrl + + : 확대
Ctrl + - : 축소
Ctrl + Y : 아웃라인 보기
Ctrl + ; : 안내선 표시/숨김

문자 작업

Ctrl + Shift + O : 아웃라인 만들기
Ctrl + Shift + C : 가운데 정렬
Ctrl + Shift + L : 왼쪽 정렬
Ctrl + Shift + R : 오른쪽 정렬
Ctrl + → : 다음 단어로
Ctrl + ← : 이전 단어로

F6 : 색상
