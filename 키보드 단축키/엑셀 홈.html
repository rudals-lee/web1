🚀 이동 및 탐색 (Ctrl)

Ctrl + Home: A1 셀로 이동
Ctrl + End: 마지막 데이터 셀
Ctrl + ← / →: 데이터 영역 끝
Ctrl + ↑ / ↓: 데이터 영역 끝
Ctrl + Page Up/Down: 워크시트 전환

🔢 셀 범위 선택 (Shift)

Shift + ↑/↓: 위아래 셀 선택
Shift + ←/→: 좌우 셀 선택
Shift + Ctrl + ←/→: 데이터 끝까지 선택
Shift + Home/End: 행 처음/끝까지 선택
Shift + Space: 행 전체 선택
Shift + Ctrl + Space: 열 전체 선택

⚡ 고급 Ctrl+Shift 조합

Ctrl+Shift+↑/↓/←/→: 데이터 끝까지 선택
Ctrl+Shift+Home: A1부터 현재 셀까지
Ctrl+Shift+End: 마지막 데이터까지 선택
Ctrl+Shift+Enter: 배열 수식 입력
Ctrl+Shift+L: 자동 필터 토글
Ctrl+Shift+8: 현재 데이터 영역 선택

🎨 Ctrl+Shift 서식 지정

Ctrl+Shift+~: 일반 숫자 서식
Ctrl+Shift+1: 숫자 서식 (소수점 2자리)
Ctrl+Shift+2: 시간 서식 (h:mm)
Ctrl+Shift+3: 날짜 서식 (d-mmm-yy)
Ctrl+Shift+4: 통화 서식
Ctrl+Shift+5: 백분율 서식
Ctrl+Shift+6: 지수 서식
Ctrl+Shift+7: 윤곽선 테두리
