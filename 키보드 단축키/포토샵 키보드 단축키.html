🎯 포토샵 단축키 학습 팁

  • 실제 작업에 바로 적용해보기
  • 한 번에 많은 단축키보다 점진적 학습

⚡ 포토샵 필수 단축키 Top 10

포토샵에서 가장 많이 사용되는 단축키들:

  1. V - 이동 도구 (가장 기본!)
  2. B - 브러시 도구
  3. Ctrl + Z - 실행 취소
  4. Ctrl + S - 저장
  5. Ctrl + T - 자유 변형
  6. [ / ] - 브러시 크기 조절
  7. M - 선택 영역 도구
  8. Ctrl + J - 레이어 복제
  9. Ctrl + D - 선택 해제
  10. Space - 임시 손 도구

💡 생산성 향상 팁

포토샵 작업 효율을 높이는 고급 팁:

  • Alt + 마우스휠: 확대/축소
  • Space + 드래그: 화면 이동
  • Ctrl + Alt + Z: 여러 번 실행 취소
  • Shift + 도구: 관련 도구 순환
  • Caps Lock: 정확한 십자선 커서

🎨 컬러 작업 팁

색상 작업을 위한 유용한 단축키:

  • D: 기본 흑백 색상
  • X: 전경/배경색 교환
  • Alt + Backspace: 전경색 채우기
  • Ctrl + Backspace: 배경색 채우기
