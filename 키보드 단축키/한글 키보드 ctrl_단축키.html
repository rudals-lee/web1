⚡ 한글 필수 단축키 Top 5

한글에서 가장 많이 사용되는 단축키들:

  1. Ctrl + S - 저장 (반드시 자주!)
  2. Ctrl + C, V - 복사, 붙여넣기
  3. Ctrl + Z - 실행 취소
  4. Ctrl + B - 굵게
  5. Ctrl + F - 찾기

💡 생산성 향상 팁

한글 문서 작성 효율을 높이는 비법:

  • Ctrl + T: 글자 모양 빠르게 변경
  • Ctrl + U: 문단 모양 설정
  • Ctrl + 3: 표 만들기로 데이터 정리
  • Ctrl + F5/F6: 시간/날짜 빠르게 삽입
