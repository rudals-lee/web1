컨닝 패드

타임라인 필수

Space : 재생/정지
B/N : 작업영역 시작/끝
Home/End : 컴포지션 시작/끝
I/O : 레이어 In/Out 이동
Page Up/Down : 1프레임 이동
Ctrl + ←/→ : 키프레임 간 이동

레이어 조작

Ctrl + D : 복제
Ctrl + Shift + D : 자르기
Enter : 이름 변경
Ctrl + L : 잠금
[ : 현재 시간으로 In 설정
] : 현재 시간으로 Out 설정

속성 보기

P : 위치 (Position)
A : 앵커포인트
S : 스케일
R : 회전
T : 투명도
U : 키프레임만 보기

키프레임

Alt + 속성 : 키프레임 생성
F9 : 이지 이즈
Ctrl + Alt + H : 속도 그래프
Ctrl + C/V : 키프레임 복사/붙여넣기
Alt + Shift + 속성 : 시간 변경

도구 & 뷰

V : 선택 도구
Y : 팬 도구
Z : 줌 도구
W : 회전 도구
Ctrl + T : 텍스트 도구
G : 펜 도구
