🎯 일러스트레이터 단축키 학습 팁

Adobe Illustrator 단축키를 효과적으로 익히는 방법:

  • 기본 선택 도구부터 시작하기
  • 펜 도구와 베지어 곡선 연습
  • 벡터 그래픽 특성 이해하기
  • 실제 로고 제작에 바로 적용

⚡ 일러스트레이터 필수 단축키 Top 10

일러스트레이터에서 가장 많이 사용되는 단축키들:

  1. V - 선택 도구 (가장 기본!)
  2. A - 직접 선택 도구
  3. P - 펜 도구
  4. T - 텍스트 도구
  5. M - 직사각형 도구
  6. Ctrl + Z - 실행 취소
  7. Ctrl + C/V - 복사/붙여넣기
  8. Ctrl + D - 변형 재적용
  9. Ctrl + G - 그룹화
  10. Space - 임시 손 도구

💡 벡터 그래픽 팁

일러스트레이터 작업 효율을 높이는 고급 팁:

  • Shift + 드래그: 비례 유지 크기 조정
  • Alt + 드래그: 복제하며 이동
  • Ctrl + Shift + O: 텍스트 아웃라인화
  • Ctrl + Y: 아웃라인/미리보기 전환
  • Ctrl + U: 스마트 가이드 토글

🎨 컬러 작업 팁

색상 작업을 위한 유용한 단축키:

  • D: 기본 채우기/선 색상
  • X: 채우기/선 색상 교환
  • ,: 채우기 없음
  • .: 선 없음
  • /: 색상 반전

📐 펜 도구 마스터 팁

일러스트레이터의 핵심 도구인 펜 도구 활용법:

  • P: 펜 도구 선택
  • Alt: 방향점 개별 조작
  • +/-: 앵커 포인트 추가/삭제
  • Shift + C: 앵커 포인트 변환
  • Ctrl + Alt + J: 패스 연결
