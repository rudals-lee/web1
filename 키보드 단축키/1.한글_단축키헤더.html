Ctrl + Shift + C가운데 정렬문단 가운데 정렬

Ctrl + N, T표 만들기새 표 삽입

Shift + Tab이전 셀표에서 이전 셀로 이동

컨닝 패드

가장 많이 쓰는 이동

Ctrl + → / ← : 단어 단위 이동
Ctrl + ↑ / ↓ : 문단 단위 이동
Home / End : 줄 처음/끝
Ctrl + Home/End : 문서 처음/끝
Page Up/Down : 화면 이동
Alt + Page Up/Down : 쪽 이동

복사 & 붙여넣기

Ctrl + C : 복사
Ctrl + X : 오려두기
Ctrl + V : 붙이기
Ctrl + Alt + V : 골라 붙이기
Shift + 방향키 : 블록 설정

서식 단축키

Ctrl + B : 글자 굵게
Ctrl + I : 글자 기울이기
Ctrl + U : 글밑줄
Shift + Alt + E/R : 글자 크기
Ctrl + M, R : 빨간색

표 편집

Ctrl + N, T : 표 만들기
Tab : 다음 셀
Shift + Tab : 이전 셀
Alt + 방향키 : 셀 크기 조정
Ctrl + Enter : 표 나누기

특수 기능

F8 : 한자 변환
Alt + F9 : 한자↔한글
F7 : 맞춤법 검사
F5 : 쪽 설정
Shift + Alt + A/Z : 줄간격
