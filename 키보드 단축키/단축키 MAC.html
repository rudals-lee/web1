즉시 화면 잠금 Ctrl + ⌘ + Q
숨김 파일 보기 ⇧ + ⌘ + .

Quick Look 미리보기 Space

🌟macOS Sonoma 주요 기능

데스크탑 위젯
위젯을 데스크탑에 직접 배치하고 iPhone 위젯과 연동 가능
발표자 오버레이
화면 공유 중 발표자 웹캠 영상을 화면에 겹쳐 표시
Safari 웹앱
웹사이트를 데스크탑 앱처럼 Dock에 고정하여 실행
게임 모드
Apple Silicon Mac에서 게임 성능 최적화 및 지연 시간 최소화
향상된 받아쓰기
텍스트 입력과 받아쓰기 동시 지원, 인식 정확도 개선
PDF 지원 강화
Notes에서 PDF 인라인 표시, 자동 양식 완성 기능

💡생산성 향상 팁

단축키 사용자화: 시스템 설정 → 키보드 → 키보드 단축키에서 나만의 단축키 설정
스크린샷 저장 위치: ⌘ + ⇧ + 5 → 옵션에서 저장 위치 변경 가능
Mission Control 활용: 데스크탑을 용도별로 구분하여 멀티태스킹 효율성 증대
Spotlight 검색: 파일, 앱, 계산기, 단위 변환 등 다양한 기능 통합 제공
트랙패드 제스처: 시스템 설정 → 트랙패드에서 다양한 제스처 활성화
