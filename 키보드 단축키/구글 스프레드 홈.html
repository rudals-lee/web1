🚀 시작하기

구글 스프레드시트의 첫 걸음

💻 접속 방법

  • Google Drive 또는 Google Sheets 접속
  • 새로 만들기 → Google 스프레드시트
  • 빠른 접속: https://sheets.new

🎯 화면 구성

  • 셀(Cell): 데이터 입력 공간
  • 행(Row): 가로줄 (1,2,3...)
  • 열(Column): 세로줄 (A,B,C...)
  • 수식 입력줄: 공식 작성 영역

✍️ 기본 입력

  • 셀 클릭 → 입력 후 Enter
  • 줄바꿈: Alt + Enter (윈도우)
  • 복사/붙여넣기: Ctrl + C / Ctrl + V

🔧 주요 기능

  • 조건부 서식으로 자동 색상 변경
  • 필터/정렬로 데이터 관리
  • 실시간 협업 가능
  • 공유 버튼으로 간편 공유

📐 기본 공식 마스터

데이터 처리를 위한 필수 공식들

💡 Quick Tip

모든 공식은 = 기호로 시작합니다!

📊 수학·통계 함수

기능 공식 예시 설명
합계 =SUM(A1:A10) A1부터 A10까지의 합
평균 =AVERAGE(B1:B5) 범위 내 평균값
최대/최소값 =MAX(C1:C10)
=MIN(C1:C10)		 최댓값, 최솟값
개수 세기 =COUNT(A1:A10) 숫자가 들어있는 셀의 개수
조건부 합계 =SUMIF(A1:A10, ">10") 조건을 만족하는 값만 합계
날짜 차이 =DATEDIF(A1, B1, "D") 두 날짜 사이의 일수 계산
오늘 날짜 =TODAY() 현재 날짜 자동 표시
현재 시간 =NOW() 현재 날짜와 시간 표시

🧠 논리 함수

기능 공식 예시 설명
조건문 =IF(A1>10, "합격", "불합격") 조건에 따라 다른 값 반환
다중 조건 =IFS(A1>=90,"A", A1>=80,"B", TRUE,"C") 여러 조건을 순차적으로 확인
AND 조건 =AND(A1>0, B1<100) 모든 조건이 참일 때 TRUE
OR 조건 =OR(A1="사과", A1="바나나") 하나라도 참이면 TRUE
오류 처리 =IFERROR(A1/B1, "계산 불가") 오류 발생 시 대체값 표시

📝 텍스트 함수

기능 공식 예시 설명
문자 연결 =CONCATENATE(A1, " ", B1) 여러 셀의 문자열 결합
텍스트 형식 =TEXT(A1, "yyyy-mm-dd") 숫자나 날짜를 특정 형식으로 변환
구분자로 결합 =JOIN(", ", A1:A5) 구분자를 사용해 여러 값 결합
텍스트 분할 =SPLIT(A1, ",") 구분자로 텍스트를 여러 셀로 분할
공백 제거 =TRIM(A1) 앞뒤 공백 및 중간 여분 공백 제거
텍스트 대체 =SUBSTITUTE(A1, "구", "신") 특정 텍스트를 다른 텍스트로 바꾸기
텍스트 비교 =EXACT(A1, B1) 두 텍스트가 정확히 일치하는지 확인

🔍 조회·참조 함수

기능 공식 예시 설명
세로 조회 =VLOOKUP(A1, B:D, 2, FALSE) 왼쪽 열에서 값을 찾아 오른쪽 값 반환
고급 조회 =XLOOKUP(A1, B:B, C:C) VLOOKUP보다 유연한 조회 함수
데이터 쿼리 =QUERY(A:C, "SELECT * WHERE B > 100") SQL과 유사한 문법으로 데이터 조회
조건 필터 =FILTER(A:C, B:B > 50) 조건에 맞는 행만 필터링
데이터 정렬 =SORT(A:C, 2, TRUE) 지정된 열 기준으로 데이터 정렬
중복 제거 =UNIQUE(A:A) 중복값을 제거하고 고유값만 반환

🔗 배열·외부 연동 함수

기능 공식 예시 설명
배열 공식 =ARRAYFORMULA(A1:A10*2) 전체 범위에 공식을 한번에 적용
행/열 바꾸기 =TRANSPOSE(A1:C3) 행과 열의 위치를 서로 바꿈
다른 시트 가져오기 =IMPORTRANGE("시트ID", "시트1!A1:B10") 다른 구글 시트에서 데이터 가져오기
웹 데이터 가져오기 =IMPORTDATA("https://example.com/data.csv") 웹상의 CSV 파일 데이터 가져오기
HTML 데이터 추출 =IMPORTXML(URL, "//table") 웹페이지에서 특정 요소 추출

  // 여기에 로직 작성
  return 값1 + 값2;
}

📧 자동 이메일 발송 예제

function sendReport() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getRange("A1:D10").getValues();
  var email = "example@gmail.com";
  
  MailApp.sendEmail(
    email, 
    "오늘의 보고서", 
    "데이터:\n" + JSON.stringify(data)
  );
}
