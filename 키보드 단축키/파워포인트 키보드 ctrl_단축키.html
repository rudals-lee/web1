🎯 PowerPoint 단축키 학습 팁

PowerPoint 단축키를 효과적으로 익히는 방법:

  • 프레젠테이션 작업 시 자주 사용하는 기능부터
  • 슬라이드 편집과 서식 단축키 우선 학습
  • 개체 작업 단축키로 디자인 효율성 향상
  • 발표 모드에서 사용하는 단축키 연습

⚡ PowerPoint 필수 단축키 Top 5

가장 많이 사용되는 단축키들:

  1. Ctrl + M - 새 슬라이드 추가
  2. Ctrl + D - 개체 복제
  3. F5 - 슬라이드쇼 시작
  4. Ctrl + G - 개체 그룹화
  5. Ctrl + S - 저장 (항상 중요!)

🎬 프레젠테이션 모드 단축키

발표 중 유용한 단축키:

  • F5: 처음부터 슬라이드쇼
  • Shift + F5: 현재 슬라이드부터
  • Esc: 슬라이드쇼 종료
  • B: 검은 화면으로 전환
  • W: 흰 화면으로 전환
