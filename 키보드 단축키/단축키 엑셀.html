함수사전
통계함수
AVERAGE =AVERAGE(A1:A10)/숫자평균 표시
AVERAGEA =AVERAGEA(A1:A10)/전체값포함평균
MIN =MIN(A1:A10)/숫자열에서 최솟값 출력
MAX =MAX(A1:A10)/숫자열에서 최댓값 출력
COUNT =COUNT(A1:A10)/숫자가 들어있는 셀의 개수 출력
COUNTA =COUNTA(A1:A10)/빈셀 제외 모든 셀 개수 출력
COUNTIF =COUNTIF(A1:A10,"조건")/조건에 맞는 셀 개수 출력
LARGE =LARGE(A1:A10,1)/지정한 숫자만큼 큰 값
SMALL =SMALL(A1:A10,1)/지정한 숫자만큼 작은 값
AND =AND(TRUE,FALSE)/모든 조건이 TRUE일때 TRUE 반환
OR =OR(A1=1;B1=2;C1=3...)/하나의 조건이 만족되면 TRUE 반환
수학 함수
SUM =SUM(A1:A10)/셀의 지정한 범위 안의 값들을 모두 더한값을 출력
SUMIF =SUMIF(A1:A10,"조건",B1:B10)/조건문을 넣어 해당하는 값만 합계
ABS =ABS(A1)/절댓값을 구해줍니다
ROUND =ROUND(A1,0)/반올림값을 구해줍니다 A1의 0자리를 반올림
ROUNDUP =ROUNDUP(A1,0)/올림값을 구해줍니다
ROUNDDOWN =ROUNDDOWN(A1,0)/내림값을 구해줍니다
MOD =MOD(10,3)/나머지를 구해줍니다
IF =IF(C1>1000,"성공","실패")/조건에 따라 값을 출력
IFERROR =IFERROR(A1/A2,1)/첫번째 수식에서 오류 시 두 번째 값을 출력
찾기 함수
HLOOKUP =HLOOKUP(찾을값, A1:C10, 3)/열범위 구간 위에있는 첫 번째 행에서 찾을값을 찾아서 지정한 행번째 값을 구해줍니다
VLOOKUP =VLOOKUP(찾을값, A1:C10, 3)/범위의 첫번째 열에서 찾을값을 검색해서 같은 행에 있는 지정열번째 값을 출력
INDEX =INDEX(A1:C10,1,3)/지정된 행과 열이 교차되는 값 출력
PV =PV(이자율, 기간 수, -연금액, 미래액, 유형(1=기초 0 =후급))
PMT =PMT(이자율, 기간수, -원금액, 미래액, 연금지급시기(1=기초 0 =후급))
